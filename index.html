Work Experience

GuildOne Inc.

Software Developer
April 2018 - Present

GuildOne Inc. is a Calgary software company that provides advanced data solutions and business intelligence to oil and gas companies. In 2018, GuildOne conducted the world’s first blockchain oil and gas royalty transaction on Energy Block Exchange (EBX), its blockchain business network for energy transactions.

GuildOne has allowed me to be part of many products such as Royalty Studio, SynMine and EBX. Being part of so many projects has allowed me to touch many parts of the programming world such as

  • OOP and functional programming
  • Unit tests
  • API testing
  • UI testing
  • Front end development
  • Blockchain development
  • Documentation

At GuildOne, I primary work on ConTracks, the world's first O&G royalty transaction. During my time on ConTracks, I was able to get Corda certified! This meant that I have a solid knowledge of blockchain concepts such as blocks, transactions, consensus, UTXO, smart contracts, public key cryptography. As well, I am able to contribute, read and maintain Corda applications. While working on ConTracks, with it has has many different moving parts, I work primarily on the front end and the middle layer where the API communicates to the blockchain layer.

The front end is primarily done in JavaFX, Oracle's software platform for delivering desktop applications. In the application, I help in creating a better user experience while maintaining functionality for users. The application is able to format tables, export data and import data.

Another new tool in my arsenal was being exposed to QA testing. I was part of a small team that tested the API where we did numerous types of testing such as validation testing, functional testing, runtime/error detection.

As part of the team working on ConTracks, I helped research in the implementation of a new web interface. Previously, ConTracks was available only as a desktop application. Since the application was written in the Scala programming language, I decided to use Scala.js. Scala.js is a compiler that compiles Scala source code to equivalent JavaScript code. With this ability to convert Scala code to JavaScript, I able to use React as our front end. This is my first exposure to React, and so far, it's easy to see why it's one of JavaScript's most popular front end frameworks!

I also help develop an internal documentation site where GuildOne employees could go and look up either business related information about a product, to technical documentation about API endpoints to blockchain concepts used. This was a great opportunity for me to learn more about pipelines that are used in BitBucket and Docker Hub. I also help maintain the technical aspect of GuildOne's website by keeping the development environment separate from the live site using an Ubuntu server.
